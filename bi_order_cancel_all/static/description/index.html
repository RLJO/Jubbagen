All in One Cancel/Reset Sale Order, Purchase Order, Picking, Invoice and Inventory Adjustment Odoo Apps

This odoo apps is complete all in one solution to cancel and reset the stocks which is transferred and reset the orders (sales/purchase/invoice) which is confirmed. This odoo module allows to cancel sales order, purchase order,any pickings like Delivery orders/Incoming shipment/Internal Transfer, Invoice and Inventory Adjustments order in confirm and Done state. Once you cancel the sales and purchase order it also cancel related invoices and receipt order /delivery orders of that particular order in odoo. After installing this Odoo apps you can easily cancel and reset the validated/done inventory adjustments as well .This odoo module also have feature of cancel the done picking and set it to draft stage.

Features

Sale Order/Purchase Order Cancel

Option of canceling the sale order/purchase order in confirm stage.

Cancel/Remove Generated Quants

Cancelled picking will also effect on generated quants.

Picking Cancel on Done stage

Option of canceling the picking in done stage.

Cancel/Remove Generated Moves

Cancelled picking will also effect on generated stock moves.

Delete picking order.

Cancel the picking the picking will be in draft stage. so you can delete that picking order.

Inventory Adjustment By Lot/Serial Number

Cancel/reset product of lot/serial number.
